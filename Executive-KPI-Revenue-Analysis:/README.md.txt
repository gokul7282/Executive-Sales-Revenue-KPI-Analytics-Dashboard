# 📊 Executive Sales, Revenue & KPI Analytics Dashboard

## 📌 Project Overview
This project is an end-to-end data analytics solution built using Python, SQL, and Power BI to analyze sales performance, revenue trends, customer behavior, product categories, and regional insights for executive decision-making.

The project follows a structured analytics workflow from data cleaning to visualization.

---

## 🛠 Tools Used
- Python (Data Cleaning & EDA)
- SQL (Data Storage & Analysis)
- Power BI (Dashboard & Visualization)

---

## 🔄 Project Workflow
- Raw sales data collected
- Data cleaned and processed using Python
- Data structured into SQL tables (Fact & Dimension tables)
- SQL used for data validation and analysis
- Data connected to Power BI for visualization

---

## 📊 Dashboard Pages
- Executive KPI Dashboard
- Sales Analysis
- Customer Analysis
- Product Analysis
- Regional Analysis

---

## 📈 Key Insights
- Technology category generated the highest revenue and profit
- APAC and United States were top performing regions
- Sales showed peak performance in later months of the year
- Revenue is concentrated in a small set of products and customers

---

## 📁 Project Structure
- Dataset (Raw + Cleaned)
- Python Scripts (Data Cleaning)
- SQL Queries
- Power BI File (.pbix)
- Business Insights (Text File)

---

## 🚀 Outcome
This project provides actionable business insights through an end-to-end data pipeline, enabling data-driven decision making in sales and performance analysis.
